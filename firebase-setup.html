🔧 ตั้งค่า Firebase

ใส่ API Key จาก Firebase Console

⚠️ ให้ดำเนินการตามขั้นตอนด้านล่างเพื่อได้รับ API Key
ขั้นตอน 1: ไปที่ Firebase Console เปิด Firebase Settings
ขั้นตอน 2: หา "Your apps" ค้นหาส่วน Your apps → เลือก Web app
ขั้นตอน 3: คัดลอก Config ข้างใต้จะมี firebaseConfig object ให้ดึงค่า:
  • apiKey
  • appId
ตัวอย่าง: AIza-1234567890abc...
ตัวอย่าง: 1:4205339157:web:ab...

🔒 ข้อมูลจะบันทึกเฉพาะในไฟล์โครงการเท่านั้น
